\# Filmorate



\## Схема базы данных



Диаграмма базы данных приложения Filmorate, отображающая структуру таблиц, связи между ними и основные поля.



\[!\[Схема базы данных Filmorate](https://dbdiagram.io/d/69c2e7b6fb2db18e3bf5d658)



\### Описание структуры



Схема состоит из следующих основных сущностей:



\- \*\*users\*\* — хранит информацию о пользователях (id, email, login, name, birthday)

\- \*\*films\*\* — содержит данные о фильмах (id, name, description, release\_date, duration, rating\_id)

\- \*\*mpa\_ratings\*\* — справочник рейтингов MPA (G, PG, PG-13, R, NC-17)

\- \*\*genres\*\* — справочник жанров (Комедия, Драма, Мультфильм и др.)

\- \*\*film\_genres\*\* — связующая таблица для связи фильмов с жанрами (many-to-many)

\- \*\*friendship\*\* — таблица для управления дружбой между пользователями (с указанием статуса подтверждения)

\- \*\*likes\*\* — хранит лайки пользователей на фильмы
